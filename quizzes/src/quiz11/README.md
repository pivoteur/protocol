# quiz11

Here we see if we can read the open-pivots (we can), but then we write out
the pivots, AND THEN we see if we can read back in those just written pivots
AND CONFIRM that the newly-read pivots match the pivots in hand.

This may entail computing equivalence between pivots. Hm.

The need for rwr comes from `virtsz` needing to write out the new open pivots,
... but other dapps are going to be doing this, as well.
