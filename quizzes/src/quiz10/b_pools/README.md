# b_pools

Reads the pools then spits out the asset-types for each pool... as a Javascript 
list. Just because.

`$ pools <auth> <date>`

where:

* `<auth>` authentication for the protocol, e.g.: `PIVOT`
* `<date>` today's date, e.g.: `$LE_DATE`
