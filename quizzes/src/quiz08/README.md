# quiz08

We read the output of `dusk` and parse it into Rust structures.

* [a_table](a_table): first we read the output of `dusk` into a table.
* [b_wyrd](b_wyrd): Converts a close-pivot call to a close pivot row
* [c_appender](c_appender): Appends the output of `wyrd` to the appropriate
close pivot table
* [d_convcls](d_convcls): Converts the old-style close pivot table to the 
new-style close pivot table using information from the open pivot table.
