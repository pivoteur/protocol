# `quotes`

![`quotes` JSON](../../quizzes/src/quiz10/d_quotes/imgs/01-quotes.png)

Prints today's crypto quotes as JSON.

## Usage

`$ quotes <date>`

where:

* `<date>` is the date to pull the crypt quotes
