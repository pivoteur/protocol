# `hwaet`

Computes the available assets for all pivot pools and reports on the health of
same.

`$ hwaet <protocol> <date>`

where:

* `<protocol>` is the pivot pool protocol, e.g.: `PIVOT`
* `<date>` is the date to evaluate the pivot pools, e.g.: `$LE_DATE`
